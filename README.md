# 🎯 BonBon Test - Ôn thi THPT

Website ôn tập trắc nghiệm Tiếng Anh với giao diện GenZ thân thiện, đặc biệt tối ưu cho điện thoại.

## ✨ Tính năng

- 📝 5 bài test Tiếng Anh + 1 bài test tổng hợp
- 🎲 Xáo trộn đáp án mỗi lần làm
- ⚡ Hiển thị kết quả ngay lập tức
- 🎈 Hiệu ứng bong bóng vui nhộn
- 📱 Tối ưu cho mobile
- ⏱️ Đếm thời gian làm bài
- 📊 Xem lại đáp án chi tiết
